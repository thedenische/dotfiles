Used utils:

- [fd](https://github.com/sharkdp/fd) - alternative to `find` command
- [fzf](https://github.com/junegunn/fzf) - command line fuzzy finder
- [jenv](https://github.com/jenv/jenv) - configuring multiple java versions
- [direnv](https://github.com/direnv/direnv) - configurin environments variables per directory
- [lcd](https://github.com/lsd-rs/lsd) - alternative to `ls` command
- [glab](https://gitlab.com/gitlab-org/cli) - GitLab cli tool
